# ACS-GEO-010: Optimizar meta/keywords en subpaginas

**Severidad:** 🟢 Medium | **Fase:** 3 | **Tiempo:** 45 min

Cada subpagina de servicio (`/servicios/[slug]`) debe tener metadata unica:
- `web-development` -> keywords: desarrollo web Chile, paginas web, e-commerce
- `business-systems` -> keywords: software a medida, sistema POS, booking online
